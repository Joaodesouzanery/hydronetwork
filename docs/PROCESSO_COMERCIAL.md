## 1. VISÃO GERAL DO PRODUTO

### O que é
Plataforma SaaS (Software as a Service) de engenharia integrada para empresas de saneamento, construtoras e consultorias. Unifica dimensionamento hidráulico, orçamentação, planejamento Lean e gestão de obra em uma única solução online.

### Proposta de Valor
> "Do levantamento topográfico ao Relatório Diário de Obra — em uma única plataforma."

**Dor do cliente:** Hoje o engenheiro usa 5-8 ferramentas separadas (Excel, AutoCAD, EPANET, MS Project, planilhas de orçamento, WhatsApp para RDO) e perde horas refazendo dados entre elas.

**Solução:** Uma plataforma que integra todos esses fluxos, eliminando retrabalho e reduzindo erros.

## 2. MÓDULOS E FUNCIONALIDADES

### NÚCLEO HYDRONETWORK (17 módulos)

| # | Módulo | O que faz | Valor para o cliente |
|---|--------|-----------|---------------------|
| 1 | **Rede de Esgoto** | Dimensionamento completo conforme NBR 9649 (Manning, verificação lâmina, tensão trativa) | Substitui planilhas Excel + cálculos manuais |

| 8 | **RDO Digital** | Relatório Diário de Obra com fotos, equipes, clima, observações | Elimina papel e WhatsApp |
| 9 | **EPANET** | Simulação hidráulica básica integrada | Sem precisar instalar software separado |
| 10 | **EPANET PRO** | Simulação avançada com visualização de resultados | Para projetos complexos |
| 11 | **SWMM** | Simulação de drenagem urbana (em desenvolvimento) | Diferencial para projetos de drenagem |
| 12 | **QGIS Export** | Exportação para QGIS (SHP, GeoJSON, KML) com dados reais | Integração GIS nativa |
| 13 | **ProjectLibre** | Exportação para ProjectLibre (alternativa gratuita ao MS Project) | Interoperabilidade |
| 14 | **Open Project** | Integração com OpenProject | Gestão de projetos expandida |
| 15 | **Peer Review** | Revisão técnica entre pares com checklist normativo | Controle de qualidade do projeto |
| 16 | **Mapa Interativo** | Visualização georreferenciada da rede com Leaflet | Visualização em tempo real da rede |
| 17 | **Topografia** | Importação CSV/DXF/SHP com detecção automática de CRS (UTM/Geo) | Início rápido do projeto |

## 5. COMO COBRAR O CUSTO DE DESENVOLVIMENTO

### O Problema
Você precisa cobrar pelo desenvolvimento contínuo da plataforma (R$ 4.000/mês+), mas não quer que pareça "salário disfarçado" para o cliente.

### A Solução: "Taxa de Tecnologia e Infraestrutura"

O valor NÃO é cobrado como "desenvolvimento" ou "salário". É embutido na estrutura de custo como **investimento em tecnologia**, que é prática padrão no mercado SaaS.

### Opção A: Embutir no Preço do Plano (RECOMENDADO)

A forma mais elegante: o custo de desenvolvimento já está dentro do preço do plano.

**Cálculo reverso:**
```
Meta de receita mensal para cobrir desenvolvimento: R$ 4.000
Preço do plano Starter: R$ 497/mês
Clientes Starter necessários: 4.000 / 497 = ~8 clientes

OU

Preço do plano Profissional: R$ 1.497/mês
Clientes Profissional necessários: 4.000 / 1.497 = ~3 clientes
```

Com 3 clientes Profissional, você cobre o custo de desenvolvimento + sobra margem.

### Opção B: Taxa de Implantação (Setup Fee)

Cobrar uma taxa única de implantação que cobre a "configuração, customização e integração":

| Porte | Taxa de Implantação | Justificativa |
|-------|---------------------|---------------|
| Pequeno (1-2 obras) | R$ 2.500 | Setup + migração de dados + treinamento |
| Médio (3-5 obras) | R$ 5.000 | Setup + migração + treinamento + customizações |
| Grande (6+ obras) | R$ 10.000+ | Setup + migração + treinamento + integrações + consultoria |

**Vantagem:** Entrada imediata de caixa + mensalidade recorrente.

### Opção C: Contrato de Manutenção Evolutiva

Para clientes que querem funcionalidades específicas:

> **"Plano de Evolução Tecnológica"**
> - R$ 4.000/mês
> - Inclui: atualizações contínuas, novos módulos, correções prioritárias, suporte dedicado
> - O cliente recebe primeiro as novidades
> - Contrato mínimo de 6 meses

**Como apresentar:**
> "Este valor cobre a infraestrutura de nuvem (servidores, banco de dados, CDN, certificados SSL), atualizações de segurança, novas funcionalidades baseadas no feedback de uso, conformidade com atualizações normativas (quando a ABNT atualiza uma NBR, nós atualizamos o cálculo), e suporte técnico prioritário."

### Opção D: Modelo Híbrido (MAIS ESTRATÉGICO)

Combine as opções para maximizar receita:

```
RECEITA POR CLIENTE:

1. Taxa de Implantação (única): R$ 3.000 - R$ 10.000
   → Justificativa: setup, migração, treinamento

2. Mensalidade do Plano: R$ 497 - R$ 1.497/mês
   → Justificativa: acesso à plataforma, armazenamento, suporte

3. [Opcional] Consultoria Lean: R$ 2.000/mês
   → Justificativa: acompanhamento LPS, análise de restrições, KPIs

4. [Opcional] Módulos Premium: R$ 300-500/mês cada
   → EPANET PRO, SWMM, Integração QGIS avançada
```

**Exemplo prático com 1 cliente médio:**
```
Setup (mês 1):        R$ 5.000
Mensalidade (x12):    R$ 1.497 x 12 = R$ 17.964
Consultoria Lean:     R$ 2.000 x 12 = R$ 24.000 (se aplicável)
                      ─────────────────────────
Total anual:          R$ 46.964
Receita mensal média: R$ 3.914/mês por cliente
```

### Como Falar com o Cliente (Scripts)

**Quando perguntar "por que esse valor?":**
> "O valor da plataforma cobre toda a infraestrutura tecnológica — servidores em nuvem, banco de dados com backup automático, cálculos conforme normas ABNT atualizadas, simulações EPANET integradas, e atualizações contínuas. É o mesmo modelo do AutoCAD, MS Project ou qualquer software profissional. A diferença é que aqui você tem tudo integrado em uma plataforma só."

**Quando perguntar "não posso usar Excel?":**
> "Pode. Mas calcular 45 trechos de rede no Excel leva 2-3 dias. Na plataforma, leva 15 minutos. Com um engenheiro custando R$ 15.000/mês, os 2 dias de trabalho manual custam R$ 1.500. A plataforma se paga no primeiro projeto."

**Quando o cliente não quer contratar desenvolvimento contínuo:**
> "Sem problema. O plano Profissional já inclui todas as atualizações. O que ofereço como Evolução Tecnológica é prioridade nas funcionalidades que VOCÊ precisa. Por exemplo, se sua equipe precisa de um relatório específico para a concessionária, ele entra na fila prioritária."

2. **Lean Construction nativo** — LPS integrado ao planejamento (raríssimo no mercado)
3. **100% online** — Sem instalação, acesso de qualquer dispositivo
4. **Normas brasileiras** — NBR 9649, NBR 12218, SINAPI, TCU — compliance automático
5. **Preço acessível** — 10x mais barato que comprar SANCAD + MS Project + EPANET separadamente
